KS Automation SMS Notification Service

This toll-free number is used by KS Automation to send automated SMS notifications generated by a privately operated home automation monitoring system.

Message Types

  • Home automation event triggers
  • Device status notifications
  • Security and monitoring alerts

Opt-In Method

Users opt in by sending a START message to the toll-free number from their mobile device. Once the opt-in message is received, the system sends automated alerts generated by the KS Automation monitoring system. Only users who have explicitly opted in will receive messages.

Message Frequency

Low volume, event-driven alerts only. Typical volume is fewer than 100 messages per month.

Opt-Out Instructions

Reply STOP to unsubscribe.
Reply HELP for assistance.
Message and data rates may apply.

Service Owner

This SMS notification service is operated by KS Automation and maintained by Kurt Stockamp. Messages are sent only to users who have opted in to receive notifications from the system.

Contact

Email: contact@ksautomation.net

Privacy Policy

Mobile phone numbers used by KS Automation are used only for sending automated alerts generated by the home automation monitoring system. Phone numbers are not shared, sold, or used for marketing purposes.
