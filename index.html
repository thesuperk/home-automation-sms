Home Automation SMS Notification Service

This toll-free number is used to send automated SMS notifications generated by a home automation monitoring system.

Message Types

  • Water leak alerts
  • Device status notifications
  • Security and monitoring alerts

Opt-In Method

Authorized users opt in by sending a START message to the toll-free number from their mobile device. Once opted in, the system sends automated alerts generated by the home automation monitoring system. Alerts are sent only to authorized mobile numbers that have opted in to receive notifications.

Message Frequency

Low volume, event-driven alerts only. Typical volume is fewer than 100 messages per month.

Opt-Out Instructions

Reply STOP to unsubscribe.
Reply HELP for assistance.
