SMS Consent and Opt-In

This page describes the SMS opt-in process for KS Automation, a service owned and operated by TCME, L.L.C.

How to Opt In

To opt in, text START to +1-888-239-0737.

Consent Disclosure

By sending START to +1-888-239-0737, you are providing express consent to receive automated SMS notifications from KS Automation (TCME, L.L.C.) related to home automation monitoring alerts.

Message Types

Messages may include security alerts, device status notifications, water leak alerts, and other home automation monitoring notifications.

Message Frequency

Message frequency varies based on system activity. Message and data rates may apply.

Opt-Out Instructions

Reply STOP to unsubscribe at any time.
Reply HELP for assistance.
